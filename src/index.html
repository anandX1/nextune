What Stream Mode Does

💀 Kill all detected background bloat
⚡ Switch CPU to High Performance (100%)

🔍 Pause Windows Search indexing (no disk spikes)

🎮 Boost OBS process priority to AboveNormal
✅ Protects: OBS, Roblox Studio, Node.js, Explorer
🔄 All changes fully reversed on "End Stream"
